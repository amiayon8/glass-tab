# GlassTab

A browser new tab page based on clean glassmorphism UI, multiple selectable search engines, and no bloatware and easy to use.

## Features

- **NASA APOD Background**
- **Clean glassmorphism UI**
- **Selectable Search Engine**
- **Selectable Wallpaper Provider**

## Search Engines

- **Google**
- **Bing**
- **DuckDuckGo**
- **Yahoo**
- **Ecosia**
- **Baidu**
- **Yandex**

## Wallpaper Providers

- **Bing**
- **NASA APOD**

## Built-in Apps

- **Instagram**
- **YouTube**
- **Slack**
- **Stardance**
- **Gmail**
- **ChatGPT**
- **Gemini**

## Stack

- Next.js
- Tailwind CSS v4
- React Icons
